# 디자인 패턴 Design Pattern

## 디자인 패턴이란?
소프트웨어 공학의 소프트웨어 설계에서 공통으로 발생하는 문제에 대해 자주 쓰니는 설계 방법을 정리한 패턴
> 💡 디자인 패턴을 참고하여 개발할 경우 개발의 **효율성**과 **유지보수성**, **운용성**이 높아지며, **프로그램의 최적화에 도움**이 된다.

3. Iterator

8. Command
